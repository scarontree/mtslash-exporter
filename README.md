# MTSLASH Exporter

在随缘居帖子页一键导出 TXT 或 EPUB。

适合用来保存楼主文、指定作者楼层，或者把长帖整理成可离线阅读的电子书。

## 功能

- 支持导出 `TXT` / `EPUB`
- 支持导出全部楼层、仅楼主、指定 `UID`
- 支持简单分章
- 支持用自定义正则识别章节标题
- 自动抓取多页帖子内容

## 安装

1. 先安装一个用户脚本管理器
   - Chrome / Edge: Tampermonkey
   - Firefox: Tampermonkey 或 Violentmonkey
2. 在 Greasy Fork 打开本脚本并点击安装
3. 确保脚本已启用

## 使用方法

1. 登录随缘居
2. 打开一个帖子正文页
3. 页面加载后，右下角会出现一个悬浮按钮
4. 点击按钮打开导出面板
5. 选择导出范围
   - `全部楼层`：导出整帖
   - `仅楼主`：只导出楼主内容
   - `指定 UID`：只导出某个用户的楼层
6. 选择导出格式
   - `TXT`
   - `EPUB`
7. 选择章节模式
   - `Chapter N`：按默认规则分章
   - `自定义正则`：使用你填写的正则识别章节标题
8. 点击“导出”

## 自定义分章

如果帖子章节标题比较规整，可以切换到 `custom` 模式，自定义正则。

示例：

```regex
^(?:第[0-9一二三四五六七八九十百千]+章|番外|后记)$
```

这类正则会按“逐行匹配”的方式识别标题。

## 适合什么场景

- 想把楼主全文保存到本地
- 只想导出某位作者或某位回复者的内容
- 想把论坛长帖整理成 EPUB 放到阅读器里看

## 注意事项

- 请先登录后再使用
- 如果帖子有权限限制、隐藏内容或中途跳转，导出可能不完整
- 个别帖子版式特殊时，分章效果可能需要手动调整正则
- 抓取页数依赖帖子分页结构，遇到异常帖欢迎反馈
